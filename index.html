Wesby - LOD Viewer

Wesby is a generic linked open data visualization tool developed with Play! Framework.

Quick start

  1. Clone Wesby repository https://github.com/weso/Wesby.git
  2. Get Typesafe Activator from Play! Framework and unzip it to a directory
  3. Run Activator and import the Wesby directory
